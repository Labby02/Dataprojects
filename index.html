Road Accident Analysis

This project was unique, because it addressed the critical global issue of road accidents which is an ongoing concern resulting in significant fatalities, injuries, and economic burdens. Using analytical methods, we conducted an in-depth analysis to identify key factors contributing to accident occurrences and severity. Our analysis also revealed temporal patterns, including accident peak days and monthly trends. Based on these insights, we developed targeted recommendations that contributed to a 15% reduction in road accidents. This Analysis was performed using Microsoft Excel.
